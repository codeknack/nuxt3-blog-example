# Nuxt 3 Minimal Blog

Example of using Nuxt 3 to accompany a tutorial on SitePoint.

## Requirements

* [Node](http://nodejs.org/)
* [Nuxt](https://v3.nuxtjs.org/)

## Installation Steps

1. Clone the repo
2. Run `npm install` to install the dependencies
3. Run `npm run dev` to start the development server
